Laravel es un marco de aplicación web elegante y poderoso diseñado para desarrolladores PHP que desean construir aplicaciones web rápidas y sofisticadas con facilidad.

Requisitos del sistema
Para ejecutar Laravel 4.4.2 en su sistema, debe cumplir con los siguientes requisitos:

PHP >= 5.4
Extensión de PHP Mcrypt
Extensión de PHP OpenSSL
Extensión de PHP Mbstring
Extensión de PHP Tokenizer
Instalación
Para instalar Laravel 4.4.2, siga los siguientes pasos:

Descargue la última versión de Laravel desde el sitio web oficial de Laravel o clone el repositorio oficial de Laravel desde GitHub.

Descomprima el archivo descargado o vaya a la carpeta del repositorio clonado.

Configure su servidor web para que la carpeta public de Laravel sea el documento raíz de su sitio.

Configure su archivo .env para la conexión a la base de datos y otras configuraciones importantes.

Ejecute el siguiente comando para instalar todas las dependencias necesarias:

Copy code
composer install
Ejecute el siguiente comando para generar una clave de aplicación única:

vbnet
Copy code
php artisan key:generate
¡Listo! Ahora puede empezar a construir su aplicación web con Laravel.
